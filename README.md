# 🛒 Smart Budget E-Commerce System

A full-stack e-commerce web application that helps users find the best products within a fixed budget. The system analyzes product prices and ratings to recommend the most valuable combination while staying under budget.

## 🚀 Features

- Budget-based shopping
- Smart product recommendations
- Price & rating optimization
- Add to Cart functionality
- Responsive design

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- Database: SQLite / MySQL

## ⚙️ Installation

```bash
git clone https://github.com/your-username/smart-budget-ecommerce.git
cd smart-budget-ecommerce
pip install -r requirements.txt
python app.py
